# ``childNode`` Mixin Methods
[Specification](https://dom.spec.whatwg.org/#interface-childnode)

The implementation of these methods are still work-in-progress.  In addition, these methods only work in browser environments; DOM is not polyfilled for Node.js.

## Module 
[`web.element.remove`](https://github.com/zloirock/core-js/blob/master/packages/core-js/modules/web.element.remove.js)

## Built-ins signatures
Those are the currently implemented features.
```ts
interface Element {
  remove(): void;
}
```

## [Entry points]({docs-version}/docs/usage#h-entry-points)
```plaintext
core-js(-pure)/stable|actual|full/element/remove
```
